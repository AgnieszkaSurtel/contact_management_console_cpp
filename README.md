# Address Book - C++ Console App

## 📄 Overview
A simple command-line address book application written in C++.  
This project was created to practice basic C++ skills such as input/output, loops, and data handling.

## 🛠️ Features
- Add a new contact
- Show all contacts
- Simple menu navigation

## 💻 Technologies
- C++
